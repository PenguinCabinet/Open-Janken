## ⬇️ インストール
```bash
go install 
```


## 🔨 使い方
1. スタートボタンを押すと、じゃんけんが始まります    
2. 「グー」、「チョキ」、「パー」ボタンを押して、出す手を選んでください
3. 「ぽん！」で固定され、勝敗の結果が表示されます
